The latest version of Woofy is 0.2.5, which you can download here. Or, if you're feeling adventurous, you can try out the latest beta version, that includes some extra features and comic definitions.

What's new since version 0.2:

  • added three new comics (Sheldon, Silent Kimbly and Wally and Osborne);
  • Woofy now has an icon, courtesy of Jeremy James;
  • added support for connecting through proxy servers;
  • rewrote most of the download logic, in order for it to be more CPU friendly;
  • added an option to minimize to the system tray;
  • added support for automatic update checks;
  • implemented error logging;
  • updated the PvPOnline definition to match the latest site changes.

In order to run it, you will need .NET Framework 2.0, which can be downloaded from the microsoft site.
