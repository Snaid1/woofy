ComicPack 0.4.2.1

2008.02.14
Download Changelog

This is the first release of ComicPack, a collection of new comics/fixes which can be deployed on top of your existing Woofy installation. This way, you won't have to download the whole package just to get some new comics, and I won't have to release a new version of Woofy just to fix some comics.

Just install it over Woofy and you're good to go.

Note that in order to run Woofy, you will need at least .NET Framework 2.0, which can be downloaded from the microsoft site.
