Woofy 0.5

2008.07.18
Download Changelog

This release includes the long awaited strip-renaming feature, so get it while it's hot! Also, when a comic has finished downloading, it is kept in the list, and rechecked at Woofy's next restart. See the changelog for more details.

Note that in order to run Woofy, you will need at least .NET Framework 2.0, which can be downloaded from the microsoft site.
